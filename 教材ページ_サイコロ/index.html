サイコロを作ろう① -HTMLでサイトの枠組みを作ろう-
教材作成日 2023年10月10日

このレッスンは、「サイコロを作ろう」シリーズの初回です。

このシリーズで最終的に完成するもののイメージは こちらです。

ボタンを押すとランダムに結果が表示されるサイコロを作ります。

初回の今回では、HTMLでサイトの枠組み部分から作っていきます。

まずは、VSCodeなどのエディタを開いてください。
下のコードをコピーして貼り付け、HTMLファイルとして保存します。

このコードでは、HTMLの基本構造を定義しています。

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>サイコロ</title>
</head>
<body>
<!-- この下にコードを書く -->
    
</body>
</html>

次に、サイコロの画像の素材を保存します。

全部で7枚の画像が必要です。
インターネット検索などで好きな画像を探すか、以下のサンプル画像を保存して使用してください。

  • デフォルト画像 : "default.png"として保存
  • 1の目 : "image1.png"として保存
  • 2の目 : "image2.png"として保存
  • 3の目 : "image3.png"として保存
  • 4の目 : "image4.png"として保存
  • 5の目 : "image5.png"として保存
  • 6の目 : "image6.png"として保存

7枚の画像は、HTMLファイルと同じフォルダに保存してください。

上の図のように、全てのファイルが同じ階層になっていればOKです。

次に、以下のように<h1>、<img>、<button>を追加します。

            <!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>サイコロ</title>
</head>
<body>
<!-- この下にコードを書く -->
<h1>サイコロ</h1>
<img src="default.png">
<button>サイコロを振る</button>

</body>
</html>
  • <h1>は、ページの見出しを表す要素です。
  • <img>は、画像を表示するための要素です。
    タグの中で、src属性に指定された "default.png" は、表示される画像のファイルパスを表します。
  • <button>は、ユーザーがクリックできるボタンを提供する要素です。

この状態で、HTMLファイルをエクスプローラーなどでダブルクリックして、ブラウザから開いてみてください。

下の図のように、見出し、デフォルト画像、ボタンが表示されていたらOKです。
（ボタンを押しても、まだ何も起きません。）

これでHTMLの枠組みができました。
